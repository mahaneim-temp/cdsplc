# CDS PLC 안드로이드 앱 설치 가이드

## 📱 설치 파일

**파일명: `cdsplc.apk`**

이 파일 하나만 휴대폰으로 전송하면 됩니다!

## 🚀 설치 방법 (매우 간단!)

### 1단계: APK 파일 받기
- 카카오톡, 텔레그램, 이메일 등으로 `cdsplc.apk` 파일을 받으세요
- 또는 USB 케이블로 휴대폰에 복사하세요

### 2단계: 파일 열기
- 파일 관리자 앱에서 `cdsplc.apk` 파일을 찾아서 탭하세요
- 또는 다운로드 폴더에서 파일을 탭하세요

### 3단계: 설치 허용
- "알 수 없는 소스에서 설치" 경고가 나오면 **"설치 허용"** 또는 **"허용"** 버튼을 탭하세요
- 처음 설치하는 경우 설정에서 허용해야 할 수 있습니다:
  - 설정 > 보안 > 알 수 없는 소스 허용
  - 또는 설치 시 나타나는 경고에서 직접 허용

### 4단계: 설치 완료
- 설치 버튼을 탭하면 자동으로 설치됩니다
- 설치 완료 후 **"열기"** 버튼을 탭하거나 앱 목록에서 **"cdsplc"** 앱을 실행하세요

## ✅ 설치 확인

앱 목록에 **"cdsplc"** 아이콘이 나타나면 설치 완료입니다!

## 🗑️ 삭제 방법

설치한 앱을 삭제하려면:
1. 설정 > 앱 > cdsplc 선택
2. "삭제" 또는 "제거" 버튼 탭
3. 확인

또는:
- 앱 목록에서 cdsplc 아이콘을 길게 누르고 "삭제" 선택

## ⚠️ 주의사항

- **인터넷 연결 불필요**: 이 앱은 오프라인에서도 작동합니다
- **Android 버전**: Android 5.0 이상 필요
- **저장 공간**: 약 10MB 정도 필요

## 🔧 문제 해결

### "알 수 없는 소스" 경고가 계속 나오면
- 설정 > 보안 > 알 수 없는 앱 설치 허용
- 또는 설치 시 나타나는 경고에서 "이 출처 허용" 체크

### 앱이 설치되지 않으면
- 저장 공간이 충분한지 확인
- Android 버전이 5.0 이상인지 확인
- 다른 APK 파일이 아닌 `cdsplc.apk` 파일인지 확인

### 앱이 실행되지 않으면
- 앱을 완전히 종료하고 다시 실행
- 휴대폰 재시작 후 다시 시도

## 📞 지원

문제가 계속되면:
- APK 파일을 다시 다운로드 받아보세요
- 휴대폰을 재시작한 후 다시 설치해보세요
